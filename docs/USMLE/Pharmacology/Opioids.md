# Classification
- Full agonists
	- Morphine, heroin
	- Methadone
	- Meperidine
	- Codeine
	- Fentanyl
- Partial agonist: buprenorphine
- Mixed agonist/antagonists
	- Butorphanol
	- Nalbuphine
	- Pentazocine
- Full antagonists
	- Naloxone
	- Naltrexone
	- Methylnaltrexone
# Adverse effects
- <span style="background:rgba(240, 200, 0, 0.2)">Constipation (not affected by tolerance, most common and persistent opioid side effect.)</span>
